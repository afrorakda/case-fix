# Case Fix
**Smart Naming Convention Converter**

A specialized utility for developers to instantly convert phrases into various programming naming conventions. Optimized for creating GitHub repository names and clean code variables.

## 1 Tool / 1 Action
- **Instant Transformation**: Type any phrase, and all major cases (Kebab, Snake, Camel, Pascal, Upper Snake) are generated simultaneously.
- **Auto-Cleaning**: Intelligently handles spaces, dots, commas, and special symbols to ensure web-safe and code-safe strings.
- **Repo-First**: "Kebab-case" is placed at the top to accelerate your GitHub project initialization.

## Features
- **Smart Parsing**: Recognizes mixed cases and various separators to extract clean word units.
- **1-Tap Copy**: Simply tap any result card to copy the value to your clipboard. 
- **Visual Feedback**: A sleek "COPIED!" overlay provides instant confirmation without cluttering the UI.

## Design Philosophy
- **Minimalist & Professional**: Pure white background with #333 main accents and 15px rounded corners.
- **Fast & Focused**: Zero-latency browser-only logic. No heavy libraries, no ads, no distractions.
- **Mobile Optimized**: Designed for quick naming brainstorms on both iPhone and Android.

## How to Install
1. Open the tool in your mobile browser.
2. **iPhone (Safari)**: Tap 'Share' -> 'Add to Home Screen'.
3. **Android (Chrome)**: Tap 'Menu (⋮)' -> 'Add to Home Screen'.
